The Code for Fizz Buzz

                        
                            // get values from the user. We need to get the fizz value and the buzz value.
                            function getValues() {
                                // get user values from the page

                                let fizzValue = document.getElementById("fizzValue").value;
                                let buzzValue = document.getElementById("buzzValue").value;

                                // parse for numbers
                                fizzValue = parseInt(fizzValue);
                                buzzValue = parseInt(buzzValue);
                                // Check entered values are integers
                                if (Number.isInteger(fizzValue) && Number.isInteger(buzzValue) && fizzValue != buzzValue) {
                                    // We call fizzbuzz
                                    let fbArray = fizzBuzz(fizzValue, buzzValue);
                                    //call displayData and write the values to the screen
                                    displayData(fbArray);
                                } else if (fizzValue == buzzValue) {
                                    alert("Please enter separate values");
                                } else {
                                    alert("Please enter integers, not decimals or fractions");
                                }
                            }


                            function fizzBuzz(fValue, bValue) {
                                let returnArray = [];

                                // loop from 1 to 100

                                //inside loop we need to check the current value in three steps
                                for (i = 1; i <= 100; i++) {
                                    // check to see if divisible by both three and five, if yes push 'FizzBuzz' into array and not the number
                                    if (i % fValue == 0 && i % bValue == 0) {
                                        returnArray.push("FizzBuzz");
                                        // check to see if divisible by fizz value (3), if yes push 'Fizz' into array and not the number         
                                    } else if (i % fValue == 0) {
                                        returnArray.push("Fizz");
                                        // check to see if divisible by fizz value (5), if yes push 'Buzz' into array and not the number
                                    } else if (i % bValue == 0) {
                                        returnArray.push("Buzz");
                                        // if none then push the number into the array
                                    } else {
                                        returnArray.push(i);
                                    }
                                }
                                return returnArray;
                            }


                            function displayData(fbArray) {

                                //get the table body element from the page
                                let tableBody = document.getElementById("results");

                                //get the template row
                                let templateRow = document.getElementById("fbTemplate");

                                //clear the table first
                                tableBody.innerHTML = "";
                                // loop over the array and create a tablerow for each item.

                                // add all the rows to the table.

                                for (let index = 0; index < fbArray.length; index += 5) {
                                    let tableRow = document.importNode(templateRow.content, true);

                                    //grab just the tds and put into array
                                    let rowCols = tableRow.querySelectorAll("td");
                                    rowCols[0].classList.add(fbArray[index]);
                                    rowCols[0].textContent = fbArray[index];
                                    rowCols[1].classList.add(fbArray[index + 1]);
                                    rowCols[1].textContent = fbArray[index + 1];
                                    rowCols[2].classList.add(fbArray[index + 2]);
                                    rowCols[2].textContent = fbArray[index + 2];
                                    rowCols[3].classList.add(fbArray[index + 3]);
                                    rowCols[3].textContent = fbArray[index + 3];
                                    rowCols[4].classList.add(fbArray[index + 4]);
                                    rowCols[4].textContent = fbArray[index + 4];

                                    tableBody.appendChild(tableRow);
                                }
                            }

The Code is structured in three functions.

Get Values

getValues stores the fizz and buzz values entered by the user. It checks to see if both numbers are different, and shows an alert if they're the same to "Please enter separate values".

It converts each string value into an integer, if the input can be converted. If the input cannot be converted to an integer, an alert is called to enter integers only.

After the inputs are converted to integers, the fizzBuzz and displayData functions are both called.

Fizz Buzz

fizzBuzz initializes an empty array to store integers. The user inputs are used as fizz and buzz parameters for a for loop.

The loop increments each index value by one. Each index value is evaluated to see if the modulus of the index divided by first, both parameters, then each parameter if the first condition is false, evaluates to zero.

If both fizz and buzz parameters are met, "FizzBuzz" is pushed to the array, instead of the index value. If not, then both the fizz and buzz parameters are checked and if one is matched, either "Fizz" or "Buzz" will be pushed. If none of these conditions are true, then the index value is pushed to the numbers array. The numbers array is returned.

Display Data

displayData gets the table body element and the template row element from the app.html page. It utilizes a template that was created in the app.html file. It lists a table row with five columns of data. The template is then initializes an empty string stored in the variable templateRows.

The page is cleared, if any data is already present, then an array is looped over and a table row is created for each item.

Each column has data and classes added from the fbArray.
